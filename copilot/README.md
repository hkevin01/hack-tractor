# Copilot Resources

This directory contains resources and configurations for GitHub Copilot integration with the Hack Tractor project.

## Purpose

- Store Copilot-specific configurations
- Maintain prompts and examples for generating consistent code
- Document AI-assisted development patterns used in the project

## Usage

When working with Copilot on this project, refer to the patterns and examples in this directory to maintain consistency in code generation and documentation.

## Contents

- `prompts/` - Example prompts for generating common code patterns
- `patterns/` - Documented code patterns preferred for this project
- `configuration/` - Copilot configuration files

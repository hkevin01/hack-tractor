# Data

This directory contains datasets and data collection utilities for the Hack Tractor project.

## Structure

- `raw/` - Raw data collected from equipment
- `processed/` - Cleaned and processed datasets
- `synthetic/` - Synthetically generated datasets for training
- `schemas/` - Data schemas and definitions

## Data Sources

- Equipment telemetry data
- Operational logs
- Environmental data (weather, soil conditions)
- Reference datasets for equipment specifications

## Usage

The data in this directory is used to train AI models and test equipment interfaces. Sensitive or large datasets are not committed to the repository - see documentation for how to obtain or generate these datasets.

# Source Code

This directory contains the source code for the Hack Tractor project.

## Structure

- `equipment/` - Libraries for interfacing with agricultural equipment
- `ai/` - AI and machine learning models for optimization and automation
- `dashboard/` - User interface components for monitoring and control

## Development Guidelines

- Follow PEP 8 style guidelines for Python code
- Document all functions and classes with docstrings
- Write unit tests for all new functionality
- Use type hints for better code readability and IDE support
